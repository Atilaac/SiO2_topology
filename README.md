# SiO2_topology
Exemplary input scripts for the generation of glass using LAMMPS and Python scripts for persistent homology analysis.

This code is used in the following publication: Achraf Atila, Yasser Bakhouch, Zhuocheng Xie, Revealing the Void-Size Distribution of Silica Glass using Persistent Homology, Materialia, 44, 102613, 2025, https://doi.org/10.1016/j.mtla.2025.102613
